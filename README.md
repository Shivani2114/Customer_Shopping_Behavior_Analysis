# Customer Shopping Behavior Analysis
Customer Shopping Behavior Analysis
# Overview

This project analyzes customer shopping behavior and sales patterns using SQL, Python, and Power BI. The goal is to extract meaningful insights about customer segments, purchasing trends, product performance, and revenue drivers to support data-driven business decisions.

The project follows a complete data analytics workflow, starting from data loading and exploration to building interactive dashboards and presenting insights through a report and presentation.

# Dataset

The dataset contains customer transaction and behavior data including:

- Customer ID

- Gender

- Age

- Product Category

- Product Name

- Purchase Amount

- Review Rating

- Shipping Type

- Subscription Status

- Previous Purchases

- Season

- Payment Method

- Discount Applied

This dataset enables analysis of customer segmentation, revenue trends, product popularity, and purchasing behavior.

# Project Workflow
# 1. Data Loading in Python

The dataset was first loaded using Python to prepare it for analysis.

Tasks performed:

- Importing dataset using Pandas

- Checking data structure

- Inspecting data types

- Understanding dataset dimensions

# 2. Exploratory Data Analysis (EDA)

EDA was conducted to understand patterns, distributions, and relationships within the data.

Key activities:

- Understanding purchase behavior

# 3. Data Cleaning

Data cleaning was performed to improve data quality.

Steps included:

- Handling missing values

- Removing duplicates

- Standardizing column formats

- Verifying data consistency

- Preparing the dataset for SQL analysis

The cleaned dataset was imported into PostgreSQL for deeper analytical queries.

Examples of analysis performed:

- Revenue by age group

- Shipping type vs purchase amount

- Discount impact on products

- Top products within each category

- Customer segmentation analysis

- Subscription behavior of repeat buyers

# Key Insights

- Young Adults generate the highest revenue, indicating strong purchasing activity.

- Clothing and Accessories are the top-performing categories.

- Most customers belong to the Loyal segment, highlighting strong retention.

- Male customers generate significantly higher revenue than female customers.

- Customers using Express shipping tend to have slightly higher purchase amounts.
